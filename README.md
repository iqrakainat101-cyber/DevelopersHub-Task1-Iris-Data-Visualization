# Iris Dataset Exploration and Visualization

## Objective
The objective of this project is to explore and visualize the Iris dataset using Python.

## Dataset
The Iris dataset is a well-known dataset containing measurements of iris flowers from three species:
- Setosa
- Versicolor
- Virginica

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-Learn
- Google Colab

## Tasks Performed
- Data Loading
- Data Inspection
- Summary Statistics
- Scatter Plot Visualization
- Histograms
- Box Plots
- Correlation Analysis

## Key Findings
- No missing values were found.
- Setosa species is clearly separable from others.
- Petal length and petal width are strongly correlated.
- Very few outliers exist.

## Conclusion
The Iris dataset is clean, balanced, and useful for learning data analysis and visualization techniques.
